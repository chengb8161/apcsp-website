Command Syntax Use Example
ls ls [OPTION]... [FILE]... List directory contents 30 ls tmp
history history (no options) Display command history 42 cp /etc/services 43 cat services 44 more services 45 less services 46 history
cd cd [directory] Changes to directory chosen 23 cd tmp
echo echo [SHORT-OPTION]... [STRING]... Echo the STRING(s) to standard output. echo “hello world”
pwd pwd [OPTION]... Print the full filename of the current working directory. 11 pwd
mkdir mkdir [OPTION]... DIRECTORY... Create the DIRECTORY(ies), if they do not already exist. 15 mkdir mydir
mv mv [OPTION]... [-T] SOURCE DEST Rename SOURCE to DEST, or move SOURCE(s) to DIRECTORY. mv tmp/readme2 readme
date date [OPTION]... [+FORMAT] Display the current time in the given FORMAT, or set the system date. date
cat cat [OPTION]... [FILE]... Concatenate FILE(s) to standard output. cat myfirstbashscript #!/bin/bash echo "Hello world"
uname uname [OPTION]... Print certain system information. uname -a Linux rpi04 4.14.79-v7+ #1159 SMP Sun Nov 4
